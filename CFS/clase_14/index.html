Anuncian lluvias para este jueves y temperaturas mínimas bajo cero para el fin de semana

Con chaparrones y lluvias aisladas desde la mañana de este jueves, el Servicio Meteorológico Nacional adelantó una jornada gris para Olavarría.

El Servicio Meteorológico Nacional (SMN) informó que se esperan lluvias aisladas y chaparrones para este jueves en Olavarría. De acuerdo al SMN, para la jornada en la ciudad se prevé una temperatura mínima de 6° C y una máxima de 12°C. Durante la madrugada estará mayormente nublado y se esperan ráfagas de viento de hasta 50 km/h. Por la mañana iniciarán las lluvias aisladas y el viento calmará, se prevén ráfagas de hasta 22 km/h. Hacia la tarde y noche continuarán los chaparrones, con una probabilidad de precipitación de hasta el 40%. Según el pronostico, afortunadamente para el viernes las precipitaciones cesarán. Sin embargo, para el sábado y domingo se anunciaron temperaturas mínimas bajo cero y ráfagas de viento de hasta 50 km/h.
